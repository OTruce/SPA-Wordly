			                    WORDLY DICTIONARY SINGLE PAGE APPLICATION(SPA)

Overview:
This is a single page application that allows users to search for English words and view their definitions, synonyms and part of speech. Additionally, it also provides an example statement for the specific word.
It communicates with Free Dictionary API to attain the required data for the user and displays results without reloading the page.

URL:
The project has been deployed using Netlify and below is the URL to access it:
			https://spa-wordly.netlify.app/
			
			
			
Features:
Allows users to search for any English word.
It displays: definitions, part of speech, example in a sentence and synonyms.
Responsive and minimalist UI.
Error handling for wrong or missing words.
No page reload needed.
